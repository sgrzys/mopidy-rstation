Music server for rSTATION

all the reasons why we chose the Mopidy as a music server for our project +
how to install the Mopidy from the source on Debian in 3 steps

we decided to use MOPIDY for our rSTATION because:

  • Mopidy server works on devices with ARM Processors,
  • it is an OpenSource software on business friendly Apache License Version 2.0,
  • it is written in Python,
  • it is very well documented,
  • it has got a vibrant community,
  • it has got more then 3000 stars on github,
  • it is extensible/hackable (Python, JSON-RPC and JavaScript APIs) -> just perfect for our project!
